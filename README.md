# App

## 기술 스택

- Yarn Berry
- TypeScript
- Vite
- Vue 2
- Vuex
- SCSS
- Vuetify 2

## 설치하기

1. VSCode Extensions 설치 (.vscode/extensions.json)
2. `yarn set version berry`
3. `yarn`
4. `yarn dlx @yarnpkg/sdks vscode`
5. TypeScript 파일을 열고 Ctrl(Cmd) + Shift + P
6. "Select TypeScript Version" 선택
7. "Use Workspace Version" 선택
