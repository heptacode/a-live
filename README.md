# App

## 기술 스택

- Yarn Berry
- TypeScript
- Vite
- Vue 2
- Vuex
- SCSS
- Vuetify 2

## 설치하기

1. VS Code Extension 설치하기 (.vscode/extensions.json)
2. `yarn`
3. `yarn dlx @yarnpkg/sdks vscode`
4. TypeScript 파일을 열고 Ctrl(Cmd) + Shift + P
5. "Select TypeScript Version" 선택
6. "Use Workspace Version" 선택
